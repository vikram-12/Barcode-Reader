# Barcode-Reader
A simple barcode reader using opencv and pyzbar .

# Requirements
1. opencv
2. pyzbar
3. pillow

# How to install?
  # openCV
  pip install opencv-python
  # pyzbar
  pip install pyzbar
  #pillow
  pip install pillow
  
 This is for windows for mac and linux please find in google.
